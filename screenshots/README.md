# Screenshots

Place your plugin screenshots here for the JetBrains Marketplace listing.

## Required Screenshots

| File | Description |
|---|---|
| `editor_table.png` | Main table editor view with translations loaded |
| `welcome_screen.png` | Welcome screen showing recent folders |
| `tools_menu.png` | Tools ▾ dropdown menu expanded |

## Tips

- Recommended width: **700–900px**
- Use the IDE's built-in screenshot tool or macOS `Cmd+Shift+4`
- Capture both light and dark themes if possible
- Make sure sample data is visible (use the `sample_arb/` folder)
